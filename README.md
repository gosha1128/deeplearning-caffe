# Installation MACOS Yosemite

## Prerequisites

* install brew for MACOS
* make sure to update your brew installation
```
brew update
```
* install openblas
```
brew install homebrew/science/openblas
```
* install nvidia toolkit
```
https://developer.nvidia.com/cuda-downloads
```
